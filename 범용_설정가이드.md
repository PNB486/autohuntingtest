# 범용 주식 일일 리포트 — 설정 및 사용 가이드

## 프로젝트 생성 (1분)

1. claude.ai → **Projects** → **+ New Project**
2. 이름: `주식 일일 리포트` (또는 원하는 이름)
3. **Instructions**에 `범용_주식리포트_시스템프롬프트.md` 전체 붙여넣기
4. 완료

---

## 빠른 시작

### 종목 등록
```
등록: SK하이닉스 000660
등록: 삼성전자 005930
등록: LG에너지솔루션 373220
등록: SanDisk SNDK
등록: 한화에어로스페이스 012450
```

### 리포트 생성 (매일 오후 3시)
```
리포트: SK하이닉스
```
→ 오늘 주가·뉴스 검색 → 업종(반도체) 자동 판별 → 밸류에이션 → 다크모드 HTML 생성

### 전체 종목 한번에
```
리포트: 전체
```
→ 등록된 모든 종목 순차 리포트 생성

## 리포트 구성 (전 종목 공통)

```
① 헤더 (종목명, 코드, 날짜)
② 메트릭 카드 4개 (주가, 적정가, 업사이드, 리스크)
③ 분기별 실적 추정 테이블
④ 분기별 적정가 차트 (Chart.js)
⑤ 리스크 변수 카드 (당일 뉴스 기반)
⑥ 시나리오 분석 (5개 + 가중 평균)
⑦ 액션 가이드 (4구간)
⑧ 핵심 모니터링 일정
⑨ 종합 판단
⑩ 푸터
```

## 커스터마이징

### 종목 추가
시스템 프롬프트 수정 없이 채팅에서 `등록:` 명령으로 즉시 추가 가능.

### 업종 모듈 추가
시스템 프롬프트의 "업종별 밸류에이션 모듈" 섹션에 새 모듈(G, H...) 추가.

### 할인율 조정
"거시 할인율 체계" 섹션의 테이블 수치 수정.

### 디자인 변경
CSS 변수(:root) 수정으로 색상 테마 일괄 변경 가능.

## 한계

- Claude 채팅을 직접 열어 명령 입력 필요 (완전 자동 아님)
- 장중 실시간 데이터는 검색 시점에 따라 지연 가능
- 종목 상태는 대화 내에서만 유지 (새 대화 시 재등록 필요)
- 미국 종목은 한국 시간 기준 전일 종가 기반
- API 비용 없음 (Pro/Team 구독 내 사용)
